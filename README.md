##### Implements a Java RMI server and client to calculate a Mandelbrot set and a traveling salesman shortest route.  
  
To Build:  
  
> ant clean  
> ant dist  
  
To Run:  
> ant runComputer  (on server machine)  
> ant runEClient [or runMClient] (on client machine)
